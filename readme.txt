== Description ==

CDN for WordPress is a simple, yet powerful WordPress plugin that is easy to use, needs minimal configuration, and best of all helps improve site performance for a faster load time. You can find more information on our web site [CDN Manager](https://cdnforwp.com/ "CDN for WordPress")

Boost your website performance globally and prevent downtime with CDN for WordPress plugin! A content delivery network is a geographically distributed group of servers optimized to deliver static content to end users.

A **content delivery network (CDN)** is a network of distributed edge servers, which accelerate your content around the globe. The main benefits of a CDN are *scalability*, *reliability* and *performance*. The **CDN for WordPress** plugin helps you to quickly and easily integrate a CDN on WordPress.

= Features =

* Fast and reliable support
* Link assets to load from a CDN (images, videos, javascripts, css, compressed files and all media files)
* Set included  directories (directories or extensions)
* Define exclusions (directories or extensions)
* Fastest upload time
* Easy and friendly usage
* Simple and basic management interface
* Easily process tracking
* Best price of the CDN market
